Project Overview

This project is an interactive analytics application that allows users to upload CSV datasets and automatically generate meaningful insights from the data.

The application performs:

Dataset profiling
Missing value analysis
Correlation detection
Summary statistics generation
Dataset health evaluation
Rule-based advanced insight generation
Dynamic dataset question answering
Downloadable analysis reports

The goal of the project is to simplify exploratory data analysis and provide quick understanding of datasets through automation.

Objective

The objective of this project is to create a beginner-friendly analytics assistant that automates repetitive exploratory data analysis tasks and helps users quickly understand the quality and patterns present in their datasets.

Project Workflow
1. Data Upload

Users upload CSV datasets through the Streamlit interface.

2. Data Processing

The uploaded dataset is processed and analyzed using Pandas and NumPy.

3. Insight Generation

The application uses rule-based logic to identify:

Missing values
Correlations
Dominant categories
Outliers
Duplicate records
Summary statistics
4. Question Answering

Users can ask dataset-related questions, and the system generates responses using keyword-based query handling.

5. Report Generation

The application generates downloadable text-based analysis reports containing:

Dataset overview
Health score
Detected issues
Generated insights
Key Insights Generated

Identify missing values and duplicate records
